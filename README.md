# agent-skills

A public collection of Agent Skills maintained by @jvm.

## What lives here
- `skills/` contains skill directories. Each skill is a folder with a required `SKILL.md` file.

## Adding a skill
1. Create `skills/<skill-name>/` using kebab-case.
2. Add a `SKILL.md` that follows the Agent Skills specification: https://agentskills.io/specification

## Current skills

| Name | Description |
| --- | --- |
| `tmux-tui-debug` | Run and debug terminal UIs inside tmux, capturing output and driving keypress interactions. |
